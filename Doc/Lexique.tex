Le language scilab comporte certains mots clefs.
Nous les regroupons par ``sémantique commune''.

\section{Les délarations de function}
Elles sont constituées des mots clefs suivants :
\begin{itemize}
\item function
\item endfunction
\end{itemize}

\section{Les boucles ``IF''}
Elles sont constituées des mots clefs suivants :
\begin{itemize}
\item if
\item then
\item else
\item elseif
\item end
\end{itemize}

\section{Les boucles ``FOR''}
Elles sont constituées des mots clefs suivants :
\begin{itemize}
\item for
\item end
\end{itemize}
